# 🏰 Добро пожаловать в CastleKeeper, путник!

Ты стоишь у ворот цифровой крепости. Здесь хранятся знания о замках, код двух стражей-приложений и мысли создателя.  
Этот vault — не просто набор файлов, а живой архив проекта **CastleKeeper**.

---

## 🗺️ Что здесь находится?

- **📦 Документация** — в папке `docs/` лежат описания архитектуры, API и базы данных.
- **📝 Заметки** — в `notes/` ты найдёшь планы, canvas-диаграммы, контракты API и вот эту самую страницу.
- **🤖 Бот** — в `bot/` живёт Java-приложение, которое слушает команды в Telegram и отвечает голосом ключника.
- **⚙️ Сервер** — в `server/` работает Spring Boot, общается с PostgreSQL и выполняет REST-запросы.
- **🗄️ База данных** — PostgreSQL 16, поднимается одной командой через Docker Compose.
- **🔧 Генератор схемы** — в `scripts/` лежит Python-скрипт, превращающий YAML-описание таблиц в Mermaid ERD-диаграмму.
- **🔧 Конфигурация** — `.env.example`, `docker-compose.yml` и другие технические файлы в корне.

---

## 🚀 Быстрый старт

1. **Изучи [[README]]** — главный файл проекта с инструкциями по запуску.
2. **Загляни в [[server]]** — чтобы понять, как устроено API.
3. **Настрой переменные окружения** — скопируй `.env.example` в `.env` и заполни секреты.
4. **Подними базу данных** — `docker compose up -d`
5. **Сгенерируй схему** — `python scripts/generate_schema.py` (или `.bat` / `.sh`)
6. **Запусти сервер** — `WIP`
7. **Запусти бота** — `WIP`
8. **Проверь в Telegram** — отправь боту `/start`

## 🤖 Команды бота

Бот управляет тремя сущностями замкового королевства:

### 🏰 Замки
| Команда | Действие |
|---------|----------|
| `/castles` | Просмотреть все замки в летописи |
| `/castle <ID>` | Найти замок по номеру |
| `/search <имя>` | Найти замок по названию |
| `/random` | Рассказать про случайный замок |
| `/addcastle` | Добавить новый замок |
| `/editcastle <ID>` | Изменить данные о замке |
| `/deletecastle <ID>` | Удалить замок из летописи |

### 👥 Авторы
| Команда | Действие |
|---------|----------|
| `/authors` | Просмотреть всех летописцев |
| `/author <ID>` | Найти летописца по номеру |
| `/searchauthor <имя>` | Найти летописца по имени |
| `/addauthor` | Добавить нового летописца |
| `/editauthor <ID>` | Изменить запись о летописце |
| `/deleteauthor <ID>` | Удалить летописца из анналов |

### 🧱 Материалы
| Команда | Действие |
|---------|----------|
| `/materials` | Просмотреть все материалы |
| `/material <ID>` | Найти материал по номеру |
| `/addmaterial` | Добавить новый материал |
| `/editmaterial <ID>` | Изменить описание материала |
| `/deletematerial <ID>` | Удалить материал из коллекции |

### 🏰 Навигация
| Команда | Действие |
|---------|----------|
| `/start` | Вернуться к вратам замка |
| `/menu` | Вернуться в главный зал |
| `/help` | Открыть фолиант со всеми командами |
