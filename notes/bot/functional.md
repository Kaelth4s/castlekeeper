Что будет позволять делать бот? (функционал)

Замки:
- Просмотреть список замков
- Посмотреть замок по ID
- Поиск замка по имени
- Добавить замок
- Удалить замок по ID
- Редактировать замок по ID
- Попросить рассказать про случайный замок

Авторы:
- Посмотреть список авторов
- Посмотреть автора по ID
- Поиск автора по имени
- Добавить автора
- Удалить автора по ID
- Редактировать автора по ID

Материалы:
- Посмотреть список материалов
- Посмотреть материал по ID
- Добавить материал
- Удалить материал по ID
- Редактировать материал по ID
