Что будет позволять делать бот? (функционал)

Условные обозначения:
↩️ — ReplyKeyboard (кнопки под полем ввода, навигация)
🔣 — InlineKeyboard (кнопки в сообщении, выбор из списка: 5 на страницу с ◀▶)
🔤 — Текстовый вывод (список, без кнопок)

Навигация:
- /start — привратник у ворот (↩️: Войти в замок, Список замков)
- /menu — главный зал с пятью крыльями (↩️: Замки, Авторы, Титулы, Материалы, Реконструкции, Выйти)
- Каждое крыло — своё подменю с reply-кнопками CRUD + Назад

Замки:
- 🔤 /castles — список замков
- 🔤 Случайный замок
- 🔣 Посмотреть (inline-список 5/page)
- 🔣 Wizard добавления (7 шагов: название, описание, автор [inline], год осн., год разр., высота, материал [inline])
- 🔣 Изменить (inline-список → выбор поля [inline] → текстовый ввод / FK [inline])
- 🔣 Удалить (inline-список → ✅/❌)

Авторы:
- 🔤 /authors — список авторов
- 🔣 Посмотреть (inline-список 5/page)
- 🔣 Wizard добавления (имя, тип автора [inline])
- 🔣 Изменить (inline-список → выбор поля имя/тип [inline] → ввод / FK [inline])
- 🔣 Удалить (inline-список → ✅/❌)

Типы авторов:
- 🔤 /author_types — список типов
- 🔣 Wizard добавления (название, описание)
- 🔣 Изменить (inline-список → ввод названия)
- 🔣 Удалить (inline-список → ✅/❌)

Материалы:
- 🔤 /materials — список материалов
- 🔣 Посмотреть (inline-список 5/page)
- 🔣 Wizard добавления (название)
- 🔣 Изменить (inline-список → ввод названия)
- 🔣 Удалить (inline-список → ✅/❌)

Реконструкции:
- 🔤 /reconstructions — список реконструкций
- 🔣 Посмотреть (inline-список 5/page)
- 🔣 Wizard добавления (замок [inline], автор [inline], год)
- 🔣 Изменить (inline-список → ввод года)
- 🔣 Удалить (inline-список → ✅/❌)

Технические детали:
- Inline-клавиатуры удаляются после использования (EditMessageReplyMarkup)
- Reply-клавиатура возвращается после завершения wizard / CRUD-операции
- DialogStateMachine отслеживает текущее подменю — кнопка «Отмена» возвращает в нужное крыло
