Что будет позволять делать бот? (функционал)

Условные обозначения:
🔤 — текстовый вывод (список без кнопок)
🔣 — inline keyboard (выбор из списка, 5 элементов на страницу с ◀▶)

Замки:
- 🔤 Просмотреть список замков
- 🔤 Попросить рассказать про случайный замок
- 🔣 Посмотреть замок (выбор из inline-списка)
- 🔣 Добавить замок (wizard: текст + выбор автора/материала из inline-списка)
- 🔣 Изменить замок (выбор замка из inline-списка → выбор поля → ввод значения)
- 🔣 Удалить замок (выбор из inline-списка → подтверждение ✅/❌)

Авторы:
- 🔤 Просмотреть список авторов
- 🔣 Посмотреть автора (выбор из inline-списка)
- 🔣 Добавить автора (wizard: имя + выбор типа автора из inline-списка)
- 🔣 Изменить автора (выбор из inline-списка → выбор поля → ввод значения)
- 🔣 Удалить автора (выбор из inline-списка → подтверждение ✅/❌)

Типы авторов:
- 🔤 Просмотреть список типов авторов
- 🔣 Добавить тип автора (wizard: название + описание)
- 🔣 Изменить тип автора (выбор из inline-списка → выбор поля → ввод значения)
- 🔣 Удалить тип автора (выбор из inline-списка → подтверждение ✅/❌)

Материалы:
- 🔤 Просмотреть список материалов
- 🔣 Посмотреть материал (выбор из inline-списка)
- 🔣 Добавить материал (wizard: название → подтверждение)
- 🔣 Изменить материал (выбор из inline-списка → ввод названия → подтверждение)
- 🔣 Удалить материал (выбор из inline-списка → подтверждение ✅/❌)

Навигация:
- /start — привратник у ворот замка
- /menu — главный зал с тремя крыльями
- /help — фолиант со всеми командами
